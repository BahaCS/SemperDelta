# Data Analyst Job Market Trends 2024

This project explores data analyst job postings across the United States in 2024 to uncover the most requested skills, tools, and experience levels.

## Project Structure
- **data/** – raw and cleaned datasets
- **notebooks/** – exploratory notebooks for scraping, cleaning, visualization, and salary analysis
- **src/** – reusable Python scripts
- **visuals/** – exported graphics used in reports

## Usage
1. Install dependencies: `pip install -r requirements.txt`
2. Optionally run `src/scrape_jobs.py` to collect job postings.
3. Use the notebooks to clean, analyze, and visualize trends.

This repository contains a small sample dataset in `data/raw/sample_jobs.csv` for demonstration.
